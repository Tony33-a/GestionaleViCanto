## Come Catturare gli Screenshot

### Browser DevTools
1. Apri l'app nel browser
2. Premi F12 per aprire DevTools
3. Premi Ctrl+Shift+P e cerca "Screenshot"
4. Seleziona "Capture full size screenshot" o "Capture node screenshot"

### Strumenti Consigliati
- **Windows**: Snipping Tool, ShareX
- **Mac**: Cmd+Shift+4
- **Cross-platform**: Lightshot, Greenshot

## Formato
- Usa **PNG** per screenshot dell'interfaccia
- Comprimi le immagini per ridurre le dimensioni del repo
- Tool consigliato: [TinyPNG](https://tinypng.com/)
